RECS|Box Redfish API v1.0.0

This is the documentation of the RECS|Box Redfish API v1.0.0.
The entry point of this API (and any Redfish API) is the ServiceRoot. One should start exploring at that Schema.

In case of an error, the API will respond with an HTTP status code as defined by the Redfish standard.
In addition to that, an error description will be returned in the format described in the Error schema.

Referenced Versions

  • DMTF Redfish Scalable Platforms Management API Specification 1.4.0
  • DMTF Redfish Schema Version 2017.3
All other referred versions, such as JSON, ODATA and SSDP, are corresponding to the normative references of the Redfish Scalable Platforms Management API Specification of the version noted above.
